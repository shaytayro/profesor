    Compare some differance between web 1.0,web 2.0, web 3.0



    1. Interaction

      1. Web 1.0 was all about static content, one way publishing of content
        without any real interation between readers or publishers or each other

      2. Web 2.0 is more about 2 way communication through social networking
        , blogging, wikis,tagging,user generated content and video

      3. Web 3.0 is curisly undefiend. AI and the web learning what you want
        and delivering you a personalized web experience

    2. Search Engine

      1. In Web 1.0 search engines retrieve macro contents.
        Search is very fast but many times results are inaccurate or more than users can chew

      2. In Web 2.0 search engines retrieve tags with micro contents
        The process of tagging is manual, tedious and covers large scale of the web
        .its tags everything: pictures, links, events, news, Blogs, audio, video, and so on.

      3. In Web 3.0 search engines will hopefully retrieve micro content texts which were tagged automatically.
        This implies translating billions of Web 1.0 macro contents into micro contents.
        The result could be more precise search because tagging can solve part of the ambiguity
        that homonyms and synonyms introduce into the process of Search.

    3. visibility And Design

      1. In Web 1.0 Pushed web, text/graphics based flash

      2. In Web 2.0 Two ways web pages, wikis, videos, pod casts, shading, Personal publishing, 2D poratals

      3. In Web 3.0 3D portals, avtar representation, Interoperable profits, Multi-user virtual environment,
        Integrated games, education and business, all media flows in and out of virtual web worlds
